# 爬虫返回值

1. 所返回的都是字典类型变量
2. 字典的`key`为排序方式`(str)`，`value`为课程的具体数据`(list)`。其中key为`default`表示爬虫默认所获取的顺序

## 课程可能的返回值：

### bilibili

- 默认排序`default`
- 名称`name`
- 播放量`plays`

### icourse163

- 默认排序`default`
- 名称`name`
- 学习人数`plays`

### icourses

- 默认排序`default`
- 名称`name`
- 学校`school`

### imooc

- 默认排序`default`
- 名称`name`
- 学习人数`plays`

### cnmooc

- 默认排序`default`
- 名称`name`
- 学校`school`
